## Overview

iNet Network Scanner Advanced discovers all devices connected to your local network and identifies their services, open ports, and network configuration, providing visibility into what is connected to your network and what services each device is running. It handles the common need to understand network composition — finding the IP address of a printer, identifying unknown devices, verifying that expected services are running, and assessing network security through port visibility.

Device discovery scans the local network and identifies every connected device with hostname, IP address, MAC address, and manufacturer identification. The scan reveals all active devices including those that do not announce themselves through standard discovery protocols. The device list provides the complete picture of network composition.

Port scanning checks for open TCP and UDP ports on discovered devices, revealing running services including web servers, SSH, file sharing, remote desktop, and other network services. The port scan identifies both expected services and potentially unwanted open ports that represent security concerns.

Service identification detects specific services running on open ports — identifying the web server software, SSH version, or file sharing protocol — providing deeper understanding of what each device is doing on the network.

For users who want to understand what is connected to their network and what services are available, iNet Network Scanner Advanced is the discovery tool that makes local network composition visible and understandable.
